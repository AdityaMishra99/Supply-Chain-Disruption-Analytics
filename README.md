# Supply-Chain-Disruption-Analytics

## Overview
End-to-end machine learning solution to predict supply chain recovery time after disruptions.  
Includes data engineering, predictive modeling, what-if simulation, resilience index, and executive dashboard deployment.

---

## Business Problem
Global supply chains face disruptions causing delays and financial losses.  
This project enables predictive recovery planning and risk assessment using machine learning.
